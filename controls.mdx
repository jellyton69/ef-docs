### Keys

-   **TAB** - Inventory
-   **Left ALT** - Activates Targeting (Third-Eye)/Toggle Focus on phone
-   **F1** - Radial Menu
-   **HOME** - Scoreboard
-   **END** - Alternate Scoreboard + Show Player IDs Above Head
-   **E** - Interact (Door Locks/Open Menus)
-   **M** - Open iPhone
-   **F11** - Cycle Voice Range
-   **F6** - Emote Menu
-   **F9** - Job Manager
-   **Y** - Clothing Wheel
-   **X** - Cancel Emote
-   **Left Control** - Crouch
-   **Left Control** - Prone/Dive (If Running)
-   **Space (While Prone)** - Flip Body
-   **B** - Toggle Point
-   **L** - Toggle Chat Visibility
-   **PAGE UP** - Scroll Chat Up (While Chat Box is Focused)
-   **PAGE DOWN** - Scroll Chat Down (While Chat Box is Focused)

### Inventory

-   **TAB** - Inventory
-   **CAPS** - Secondary Inventories (Trunk/Glovebox)
-   **RMB** - Open item context menu
-   **Z** - Toggle Inventory Key Bind Slots
-   **1-6** - Key Bind Inventory Slots
-   **ALT + RMB** - Quick use item
-   **CTRL + LMB** - Fast move a stack of items into another inventory
-   **SHIFT + Drag** - Split item quantity in half
-   **CTRL + SHIFT + LMB** - Fast move half a stack of items into another inventory

-   **/scene [Scene Text]** - Open the menu to theme and create a 3D scene in the world. (Notes/Ads/Graffiti/Etc)
-   **/hudsettings** - Open up the HUD Settings UI (Edit Layout/Disable Components)
-   **/hu** - Toggle Hands Up
-   **/givecontact** - Give contact info to nearby player

-   **/911** - Contact Emergency Services
-   **/311** - Call for Roadside Assistance
-   **/911a** - Contact Emergency Services (Anonymously)
-   **/311a** - Call for Roadside Assistance (Anonymously)
-   **/radio** - Open Sonoran Radio UI
-   **/report** - Create a support request or player report
-   **/canceljob** - Cancel a Lumberjack Job
-   **/fixjobvehicle** - Fix a vehicleâ€™s position (Lumberjack job only)
