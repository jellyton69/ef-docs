### Keys

-   **TAB** - _Inventory_
-   **Left ALT** - _Activates Targeting (Third-Eye)/Toggle Focus on phone_
-   **F1** - _Radial Menu_
-   **HOME** - _Scoreboard_
-   **END** - Alternate _Scoreboard + Show Player IDs Above Head_
-   **E** - _Interact (Door Locks/Open Menus)_
-   **M** - _Open iPhone_
-   **F11** - _Cycle Voice Range_
-   **F6** - _Emote Menu_
-   **F9** - _Job Manager_
-   **Y** - _Clothing Wheel_
-   **X** - _Cancel Emote_
-   **Left Control** - _Crouch_
-   **Left Control** - _Prone/Dive (If Running)_
-   **Space (While Prone)** - _Flip Body_
-   **B** - _Toggle Point_
-   **L** - _Toggle Chat Visibility_
-   **PAGE UP** - _Scroll Chat Up (While Chat Box is Focused)_
-   **PAGE DOWN** - _Scroll Chat Down (While Chat Box is Focused)_

### Inventory

-   **TAB** - _Inventory_
-   **CAPS** - _Secondary Inventories (Trunk/Glovebox)_
-   **RMB** - Open item context menu
-   **Z** - _Toggle Inventory Key Bind Slots_
-   **1-6** - _Key Bind Inventory Slots_
-   **ALT + RMB** - Quick use item
-   **CTRL + LMB** - Fast move a stack of items into another inventory
-   **SHIFT + Drag** - Split item quantity in half
-   **CTRL + SHIFT + LMB** - Fast move half a stack of items into another inventory
